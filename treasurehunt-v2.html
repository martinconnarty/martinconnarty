⚓ How It Works

A physical coordinate treasure hunt — players walk to real-world locations, collect letters, and unscramble a hidden word.

🗺
Upload a map
Take a screenshot from Google Maps, Apple Maps, or scan an OS map. Upload it as your hunt area.
📐
Set the grid
Drag and resize the yellow grid overlay to align with your map. Set how many columns and rows, and where the coordinates start. Use 6-figure mode if you're working from an OS map — players pinpoint a precise spot within each square.
🔤
Add words
Each word is a separate level. Players must find every letter for the current word before the next one unlocks. Use short, clear words.
⚙️
Generate coordinates
The app assigns a random grid coordinate to each letter. You can edit them manually if needed. Cells marked red (don't use) or orange (prefer not) are skipped.
🖨
Print maps
Two pages are generated: an Admin sheet showing letters at their positions (for you to set up the hunt), and a Participant map with just the blank grid for players to navigate with.
🥾
Hide the letters
Visit each coordinate and leave a physical letter clue — a card, sticker, chalk mark — at that exact spot.
🚀
Launch & play
Add players, hand out participant maps, and let them go. Each player or team tracks their own progress. When all letters for a word are found, they unscramble it to move to the next level.
🔒
Admin mode
The Admin button in the top corner locks and unlocks the builder, config, and player management tools. Set a password in Step 4 to protect it. The app always starts in admin mode so you can set up without a password prompt.
